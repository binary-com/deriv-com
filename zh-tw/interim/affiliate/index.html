探索 Deriv.com

Binary.com 已更名為 Deriv.com ，以下是其對合作夥伴的意義

如您所知，我們已經將 Binary.com 更名為 Deriv.com。這是我們最新承諾的一部分，通過簡單、靈活和可靠的解決方案，使所有人都可以方便使用及輕鬆進行線上交易。

作為我們與您的客戶的重要合作夥伴網絡的一部分，您可能很好奇並且擔心這種轉變的意義。我們整理了本指南，以幫助您更加了解品牌重塑，並為您與客戶的交流提供支持。

品牌重塑的內容是什麼？

品牌重塑活動使我們更堅定地為客戶提供靈活、便捷和信心，讓客戶能隨心所欲地交易。這是我們公司為客戶提供優異服務的第三個十年成長過程的一部分。

為什麼 Binary.com 要重塑品牌？

我們在此市場已經有20年，覺得已經到了對客戶的承諾給予更新的時候了。品牌重塑增強了我們為客戶提供簡單、靈活和可靠的線上交易解決方案的決心。

Binary.com 是否已被另一家公司收購？

不，Deriv.com 是由為您帶來 Binary.com 的同一團隊構建、擁有和管理的。20年的經驗讓我們對為您服務的能力感到從未有過的強大或自信。

Binary.com 會怎樣?

我們會將重點完全放在 Deriv.com 上，Binary.com 網站將逐步淘汰。我們鼓勵大家立即移至 Deriv.com 及享受新的體驗。客戶可在 Deriv.com 使用和存取儲存在 Binary.com 的客戶帳戶、資金以及所有內容。

推出了哪些變更？

品牌重塑涉及新名稱、徽標、公司顏色和網站。此外，Deriv.com 為客戶提供了多種工具，可在 Binary.com 的熱門平台 SmartTrader，以及 DTrader、DBot、DMT5 等四大高級平台交易。一些新功能將分階段提供。

它將如何影響我的客戶？

我們知道客戶可能會因為此轉換而措手不及。因此，我們努力確保以無縫銜接的方式轉換成 Deriv.com。作為聯盟會員，您在支持他們轉換的過程中扮演著關鍵角色。以下是他們可能提出的一些問題，以及您提供信心保證的答复。

為何我必須轉移至 Deriv.com?

品牌重塑活動由我們的客戶，就是您所啟發。為您服務20年以來，我們覺得能更加了解您的需求。品牌重塑是我們實施改進的線上交易解決方案，以滿足您所有需求的機會。

Deriv.com 有什麼新增內容?

Deriv.com 在簡單、靈活和可靠的環境中為您提供多種交易選項。除了 Binary.com 熱門的交易平台 SmartTrader 以外，它還提供 DTrader、DBot 和 DMT5 這三大核心產品。不久以後，以下新功能將會推出:

  • 乘數期權可以為交易者提供更高的利潤、同時限制風險

  • P2P 收銀台，促進無縫金融交易

  • 亮和暗模式功能，帶來更愉快的體驗

我可否仍在 Binary.com 交易?

您的資金將同時在 Binary.com 和 Deriv.com 帳戶中顯示，因為兩者本質上是同一帳戶。但是，我們建議您立即試用 Deriv.com ，因為我們將逐步淘汰 Binary.com。我們向您保證，您最喜歡的功能仍可以在 Deriv.com 使用。另外，一旦您使用 Binary.com 憑據登入，您在 Binary.com 的帳戶詳細信息、資金和自定義設定將可在 Deriv.com 使用。

如何開始使用 Deriv.com?

很簡單。只需前往 Deriv.com 並輸入現有的 Binary.com 憑據即可。您將立即能夠探索 Deriv.com 的新外觀、感受和新增功能。有許多教育資源可幫助您在新的交易空間輕鬆導航。為了增強您對新環境的信心，請使用模擬帳戶探索 Deriv.com。

它將如何轉換我們的合夥關係？

作為我們重要的合作夥伴，您無疑會擔心自身與 Deriv.com 的合作夥伴關係。請不用擔心，我們會盡力為您提供毫無影響的過渡，以便您可以繼續建立客戶網絡並擴大盈利潛能。

我的聯盟會員帳戶會怎樣？

您的 Binary.com 聯盟會員帳戶在 Deriv.com 也可使用。當您的客戶在 Binary.com 或 Deriv.com 交易時，您將可以獲得佣金。

Deriv.com 的佣金與 Binary.com 的佣金如何區分？

為了您的方便起見，我們將報告分開提供。

要獲取每個報告，請轉到儀表板，然後選擇報告>按品牌過濾。然後選擇您要查看其佣金的平台。

作為介紹經紀人（IB），此轉變將給我帶來哪些影響？

無論客戶在 Binary.com 的 MetaTrader （MT5）平台或是 Deriv.com 的 MetaTrader5（DMT5）平台交易，您都將繼續​​賺取每日佣金。

Deriv.com 是否支持 API 開發人員？

是。如果您已經使用 Binary.com API 開發了自己的交易應用程式，您仍然可以繼續使用它。您還可以使用自己的應用收取在 Deriv.com 進行的任何交易的佣金。這些佣金將在您的 Deriv.com 帳戶中顯示。

Deriv.com 是否有付款代理計劃?

是。如果您是 Binary.com 的註冊付款代理，那麼在 Deriv.com 交易的客戶將可以使用您的服務。

我要如何開始推廣 Deriv.com?

請放心，您我雙方的合作夥伴關係在此品牌重塑過程中至關重要。實際上，我們認為這是您與客戶群增強關係的大好機會。且您可以指望我們對您的努力做出響應並給予支持。

我可否立刻開始推廣 Deriv.com？

的確。雖然 Binary.com 仍然將存在一段時間，但它會逐漸淘汰。鼓勵您的客戶轉移到 Deriv.com，並讓他們在我們新的和改進的平台上交易，從而賺取佣金。

我要如何推廣 Deriv.com?

如要推廣 Deriv.com，您需要有 Deriv.com 聯盟連結。

首先登入Binary.com 聯盟會員儀表板，以獲取連結。

選擇行銷 > 獲取媒體代碼。然後複製所提供的登陸頁面 URL。

登陸頁面 URL是您專用的聯盟連結，您可用它對您網絡內的客戶推廣 Deriv.com。

你們提供哪些行銷支持？

我們非常希望您開始將 Deriv.com 品牌整合到您的所有行銷工作中。您可以前往聯盟會員儀表板使用我們提供的即用型標語和徽標，以便您能更輕鬆地開始進行工作。無論您最愛的行銷通道是什麼，您都會發現 Deriv.com 標語和徽標的格式符合您的特定需求。開始在您的所有通道使用這些工具，並透過新的 Deriv.com 品牌廣為宣傳。
